## Использование
Запуск проекта:

Склонируйте репозиторий:
```sh
$ git clone https://github.com/snow1x-blip/panorama_uwu-3/
```

создайте .env и добавьте туда следующие данные:
```
Sekret_Key=...
API_Key=...
```

разверните в отдельном докере presenton:
```sh
sudo docker start presenton
```
